# prac-react

This repository is now configured as a Next.js app.

## Run locally

1. Install dependencies:

```bash
cd /workspaces/prac-react
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the URL shown in the terminal (usually `http://localhost:3000`).

The Todo app lives in `src/app/page.js`.
